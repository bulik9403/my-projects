# Spaceship Titanic - Kaggle Competition Solution
## Results

- **Best Model:** VotingClassifier with soft voting

## Project Overview
Predict which passengers were transported to an alternate dimension during the Spaceship Titanic's collision with a spacetime anomaly. This project demonstrates end-to-end machine learning workflow from exploratory data analysis to model deployment.
